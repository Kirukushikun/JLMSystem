JL Monitoring System

Digital Workflow for Justification Letter Form Management

BFC Group  ·  IT Department  ·  2026

Overview

What is the JL Monitoring System?

A unified platform that brings JL form submission, approval, and tracking together — replacing a scattered workflow of Google Forms, Excel sheets, and Viber messages.

The Old Way

  • 📋
    Forms submitted via Google Form — but no approval chain attached
  • 💬
    Review results communicated through Viber, easy to miss or lose
  • 📊
    Approvals recorded manually in Excel, managed per person
  • 🔢
    Serial numbers assigned manually — no standard system

Now, In One System

  • ✅
    Same familiar online form — now directly feeds the approval chain
  • ✅
    Reviewers and VP act on forms inside the system — no Viber needed
  • ✅
    Live dashboard replaces the Excel sheet
  • ✅
    Serial number auto-generated on VP approval
Workflow

How It Works — 3-Step Approval Process

1

Submission

Farm manager or supervisor fills in the JL form online. No account required — just the link.

Public Access
2

Review

Reviewer checks the form details and marks it as Reviewed or rejects it with a reason.

Reviewer Role
3

VP Approval

VP Approver gives final sign-off. Upon approval, a unique serial number is automatically assigned.

VP Role
Serial Number Format: BFC-JL-001-2026 — auto-generated per company, per year, on VP approval
Features

Key System Features

🔗

Same Familiar Link

Submit from any device with just a shared link — no account needed, just like Google Form but now connected to the approval chain.

📊

Approval Dashboard

Replaces the Excel sheet and Viber updates — all forms, their status, and actions are visible in one live dashboard.

🔖

Auto Serial Numbers

No more manual code assignment — the system generates a unique serial number the moment the VP approves.

📎

Attachments Included

Supporting documents are uploaded with the form and stored alongside it — no separate sharing or file hunting.

🔍

Instant Search & Filter

All submissions live in one place — search by title, company, or manager and filter by status in seconds.

🕓

Complete History

Full timeline of every form — who submitted, who reviewed, who approved, and when. No more chasing records.

Access Control

User Roles & Permissions

👁️

Reviewer

  • View all submitted JL forms
  • Mark forms as Reviewed
  • Reject forms with a reason
  • Share form submission link
  • View attachments
✅

VP Approver

  • View reviewed forms only
  • Give final approval
  • Reject with a reason
  • Triggers serial number generation
  • View attachments
⚙️

Admin

  • Grant & revoke user access
  • Manage company/farm list
  • Manage department list
  • Full system oversight

Note: Each user has exactly one role. Access is granted by an Admin through the User Management panel — no direct database access needed.

Admin Panel

Admin Capabilities

User Management

  • 🏢
    Pulls all organization employees from the central HR system automatically
  • ✔
    Grant access with a single click — assign Reviewer, VP, or Admin role
  • ✘
    Revoke access instantly at any time
  • 📋
    See which employees have access and which don't

Maintenance

  • 🏭
    Add or remove Company / Farm entries shown in the submission form
  • 🏷
    Each company has a custom serial code used in serial number generation
  • 📂
    Add or remove Department options in the form
  • 🔒
    Deletion blocked if company/dept has existing JL entries
Security

Security & Authentication

🔐

Organization Single Sign-On

Credentials are validated against the organization's central Auth API — no separate passwords stored in this system.

🛡

Brute-Force Protection

3 failed login attempts trigger a 15-minute lockout. Attempt counter resets automatically on successful login.

📝

Access Logging

Every login attempt — successful or failed — is logged with the IP address, timestamp, and user agent.

🔒

Role-Based Route Guards

Protected pages enforce role checks at the server level. File attachments require authentication to access.
